# GBIS KRISTUS GEMBALA PEKANBARU - Website siap pakai

Ini adalah paket website statis sederhana untuk gereja Anda. Cocok untuk di-host di GitHub Pages, Netlify, Vercel, atau hosting statis lainnya.

## Isi file
- `index.html` — Halaman utama
- `styles.css` — Gaya
- `script.js` — Interaksi (kalender, pengumuman demo, galeri)
- `images/` — Contoh gambar (placeholder)
- `README.md` — Petunjuk ini

## Cara deploy cepat (GitHub Pages)
1. Buat repository baru di GitHub.
2. Upload isi folder ini ke repository.
3. Aktifkan GitHub Pages di Settings → Pages → pilih branch `main` (atau `gh-pages`) dan folder `/root`.
4. Setelah beberapa menit, situs akan tersedia di `https://<username>.github.io/<repo>/`

## Catatan
- Kalender dan pengumuman sekarang hanya berjalan secara lokal (JavaScript). Untuk menyimpan perubahan secara permanen, gunakan backend atau CMS (mis. Google Sheets + AppScript, Firebase, atau Netlify CMS).
- Untuk mengganti foto, timpa file di folder `images/` dengan nama yang sama atau sunting `index.html`.
